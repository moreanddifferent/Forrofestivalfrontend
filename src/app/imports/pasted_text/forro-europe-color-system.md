USAGE RULES
Do not use all colors equally.
The site should remain elegant and curated.

Visual balance:
- mostly warm neutrals
- blue as the main action color
- green for positive availability states
- yellow for underline and warm highlight accents
- coral only for small moments of urgency or warm emphasis
- aqua as a secondary fresh supporting accent

BACKGROUND SYSTEM
Replace cold greys with warmer neutrals:
- main page backgrounds should use Warm White (#FCFBF7)
- subtle section panels can use Soft Sand (#F5F1E8)
- borders and separators should use Light Border (#E7E0D4)

This is important: the new neutral system should make the whole interface feel less flat and less sad.

APPLY TO THE WEBSITE
Homepage
- keep blue for active nav, search actions, links, selected states
- use yellow for title underlines and small emphasis accents
- use green for “Tickets available”
- use warm white and soft sand backgrounds instead of plain cold grey

Saved pages
- keep the pages calm and clean
- use blue for selected cards or active map states
- use green for positive ticket badges
- use yellow for “Tickets not announced”
- use coral only for destructive / remove actions or small alert moments

Map
- active marker should use blue
- standard positive marker can use green
- aqua can be used for secondary map accents or legend accents
- keep the map visually soft, clean, and readable

Shared read-only page
- same overall color system
- calm, curated, minimal
- no heavy product colors everywhere

COMPONENT STYLING
Buttons
- primary buttons = blue background with white text
- secondary buttons = warm white or soft sand with light border
- destructive actions can use coral text or coral border very sparingly

Badges
- upcoming / opens soon = blue text on pale blue-tinted background or outlined blue
- available = green text on green soft background
- not announced = yellow accent with soft yellow background
- alerts / warnings = coral text on coral soft background only when truly needed

TYPOGRAPHY FEEL
Keep typography strong, clean, and high contrast.
Primary text should use #1F1F1C.
Secondary metadata should use #6B6B63.

IMPORTANT
Do not redesign layout structure from scratch.
This is primarily a visual system refinement:
- warmer backgrounds
- brighter supporting accents
- more joyful energy
- still clean and premium

The result should feel like a polished digital festival planner, not a generic SaaS dashboard.
