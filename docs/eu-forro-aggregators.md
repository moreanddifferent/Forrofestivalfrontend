# Representative Forró festival/event aggregators (Europe)

1. **Forró Federation**  
   The Forró Federation offers a curated list of upcoming festivals, marathons, and weekends across Europe and worldwide. The site aggregates event information from various sources and collaborates with several event calendars.  
   URL: https://forrofederation.com/forro-events/forro-festivals/  
   Related: https://forrofederation.com/projects-collaborative-event-lists/

2. **The Forró App**  
   This open-source community project lists Forró festivals, parties, and classes, and events appear both in the mobile app (iOS/Android) and the web interface. Festival organizers can add their events directly.  
   URL: https://forro-app.com/forro-festivals/

3. **Forró Festivals (forrofestivals.com)**  
   A dedicated platform listing upcoming Forró events around the world, with specific details for festivals in Europe. The search function allows filtering by continent, type, and year.  
   URL: https://www.forrofestivals.com/

4. **Forró pelo mundo**  
   Provides a list of Forró festivals globally, including schedules and brief event descriptions, with several featured in European cities.  
   URL: https://forropelomundo.com/festivals

5. **Forró Events**  
   A platform to discover and promote Forró events, including European festivals, and offers resources for both dancers and organizers.  
   URL: https://forro.events/  
   Events page: https://forro.events/events/

6. **Forró Foundations – Worldwide Forró Festivals Listing**  
   Features a calendar of worldwide Forró festivals, with strong coverage in Europe and detailed listings for various cities.  
   URL: https://www.forrofoundations.com/festivals

7. **Europe Festivals Calendar – Forrózin Freiburg**  
   Lists European Forró festivals month-by-month, maintaining an up-to-date calendar with links and details.  
   URL: https://forrozinfreiburg.de/tanzen-forro-festivals/
